# Readme
支出トラッカーの仕様

+ サイドメニューを押下時に全て、収入、支出のデータをDBから取得する
+ ソート機能の実装(ヘッダー押下)
+ 特定のキーワードでフィルター(未実装)
+ 収入、支出データの登録・編集機能(未実装)
+ dynamicIO/PPRなどのNextJs新機能のお試し、スケルトンUIの作成

# 利用技術
+ NextJS
+ TypeScript
+ Tailwind
+ shadcn/ui
+ supabase

# 環境構築
```
# 関連するライブラリのインストール
$ npm i

# jsonサーバの起動
$ npm run json

# 実行
$ npm run dev
```
